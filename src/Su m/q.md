## 题目描述
Given n
, please calculate S(n)
 where
S(n)=∑i=1ni∗(i+1)/2
## 输入
The 1st line is a positive integer T(1⩽T⩽100000)
which is the number of test cases.

Then T
 lines follow. Each line has an integer n(1⩽n⩽1000000)
 for a test case.
## 输出
Output T lines. Each line is the result value ans.

2
1
2
## 样例输出
1
4
